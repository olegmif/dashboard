# M3 — Строка иконок состояния системы

**Цель:** горизонтальный ряд SVG-иконок, каждая со своим состоянием и цветом по
порогу.

## Фазы

### Ph3.1 — SVG-хелпер
- `loadSVGIcon(path, size, color)` — рендер librsvg через GdkPixbuf,
  перекраска (подстановка `fill` в текст SVG или CSS `color` для symbolic).

### Ph3.2 — Layout ряда иконок
- Горизонтальный бокс, заглушки иконок.

### Ph3.3 — ОЗУ и ЦП
- Свободная ОЗУ: `/proc/meminfo`, опрос ~2 с.
- Нагрузка ЦП: `/proc/stat`, дельта между опросами, ~1–2 с.

### Ph3.4 — Раскладка клавиатуры
- `setxkbmap -query` или XKB-событие; обновление по событию/1 с.

### Ph3.5 — VPN
- Детект: `wg show` / наличие tun-интерфейса (`ip link`), ~5–10 с.
- ⚠️ Зависит от открытого вопроса: какой именно VPN.

### Ph3.6 — Пороговая раскраска
- Зелёный/жёлтый/красный по порогам из конфига, в одном месте.

## Критерий готовности
Ряд иконок отражает живое состояние системы, цвета меняются по порогам.

## Вне объёма (опционально)
- Температура (`sensors` / `/sys/class/hwmon`) — добавить при желании.
