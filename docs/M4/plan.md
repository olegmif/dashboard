# M4 — Погода

**Цель:** виджет текущей погоды + краткий прогноз без API-ключа.

## Фазы

### Ph4.1 — Клиент Open-Meteo
- HTTP-запрос к Open-Meteo (без ключа, JSON), координаты из `config.toml`.
- Запасной источник на случай недоступности — `wttr.in`.

### Ph4.2 — Опрос
- Горутина: poll раз в 10–30 мин (погода не меняется ежесекундно).

### Ph4.3 — Иконки по weather-коду
- SVG-набор ☀/☁/🌧 и т.п., маппинг weather-code → иконка.

### Ph4.4 — Отображение
- Текущая температура/состояние + краткий прогноз.

## Критерий готовности
Виджет показывает актуальную погоду, обновляется по таймеру, иконка
соответствует состоянию.
