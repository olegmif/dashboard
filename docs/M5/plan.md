# M5 — Telegram: базовый (логин → poll → карточки)

**Цель:** показать последние сообщения 3–4 диалогов карточками. Самая
трудоёмкая часть проекта — берём минимальным рабочим срезом (poll), реалтайм
в M6.

> ⚠️ Зависит от открытого вопроса: контакты — личные чаты, группы или каналы?
> От этого зависит, какие сущности слушать.

## Фазы

### Ф5.1 — Конфиг и секреты
- `api_id` / `api_hash` — в `config.toml`/env, НЕ в гит.
- session-файл и конфиг — в `.gitignore`.

### Ф5.2 — Логин-сессия
- `gotd/td`, авторизация от пользователя (номер + код).
- Сохранение сессии в файл — логинимся один раз.

### Ф5.3 — Poll сообщений
- Тянуть последние N сообщений для каждого из 3–4 диалогов.
- Опрос с щадящим интервалом (10–30 с).

### Ф5.4 — Карточка диалога
- Виджет на контакт: имя + последние N сообщений.

## Критерий готовности
После единичного логина 3–4 карточки показывают свежие сообщения, обновляются
по таймеру.

## Вне объёма (перенесено в M6)
- Стриминг апдейтов, аватары, прокрутка, переподключения.
