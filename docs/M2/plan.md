# M2 — Часы + фоторамка

**Цель:** два тривиальных виджета, дающих быстрый видимый результат на готовом
каркасе M1.

## Фазы

### Ph2.1 — Часы
- Горутина-тик раз в секунду → `glib.IdleAdd` обновляет `Label`.
- Шрифт через CSS (моноширинный для цифр).
- Второй ряд: дата / день недели.

### Ph2.2 — Фоторамка: источник
- Чтение одного/нескольких каталогов из `config.toml`.
- Сбор списка файлов-изображений.

### Ph2.3 — Фоторамка: ротатор
- Горутина раз в N сек выбирает следующее фото (последовательно/случайно).
- Шлёт путь в UI через канал.

### Ph2.4 — Фоторамка: ресайз
- `GdkPixbuf.NewFromFileAtScale(path, w, h, preserveAspect)` — ресайз ДО показа
  (фото бывают 4000×3000, иначе спайки CPU/RAM на декоде).

## Критерий готовности
Часы идут, фото сменяются по таймеру без заметных спайков нагрузки.

## Вне объёма (перенесено)
- Плавный кросс-фейд фото → M7.
