# M7 — Полировка

**Цель:** довести до ежедневного использования.

## Фазы

### Ф7.1 — Типографика
- Подбор шрифтов, размеров, отступов по всем виджетам.

### Ф7.2 — Кросс-фейд фото
- Плавная смена изображений в фоторамке (перенесено из M2).

### Ф7.3 — Плавность и производительность
- Профилирование, выверка интервалов опроса, устранение спайков.

### Ф7.4 — Автозапуск
- Запуск из `.xinitrc` / autostart dwm, корректное поведение при рестарте WM.

## Критерий готовности
Дашборд стартует сам, выглядит цельно, работает плавно при повседневном
использовании.
